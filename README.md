# Task 1: Exploring and Visualizing the Iris Dataset

## Objective

To understand how to load, explore, summarize, and visualize a dataset using Python.

## Dataset

Iris Dataset (classic dataset used for data analysis and machine learning practice).

## Approach

1. Loaded the dataset using **pandas**.
2. Examined dataset structure using:

   * `.shape`
   * `.columns`
   * `.head()`
3. Performed exploratory data analysis.
4. Created visualizations using **matplotlib** and **seaborn**:

   * Scatter plots to analyze relationships between features.
   * Histograms to examine feature distributions.
   * Box plots to detect outliers and variability.

## Key Skills Applied

* Data loading and inspection
* Exploratory Data Analysis (EDA)
* Data visualization

## Insights

* Observed relationships between sepal length, petal length, and species.
* Identified distribution patterns across features.

# Task 3: Customer Churn Prediction

## Objective

Identify customers who are likely to leave the bank.

## Dataset

Churn Modelling Dataset.

## Approach

1. Cleaned and prepared the dataset.
2. Encoded categorical features:

   * Geography
   * Gender
3. Split the dataset into training and testing sets.
4. Trained classification models.
5. Analyzed **feature importance** to determine the most influential variables affecting churn.

## Key Skills Applied

* Categorical encoding (Label Encoding / One-Hot Encoding)
* Classification modeling
* Feature importance analysis

## Insights

* Customer behavior patterns and account characteristics significantly influence churn probability.
